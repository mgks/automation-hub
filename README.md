# Automation Hub

> The universal, searchable, and hosted directory for automation workflows across n8n, OpenClaw, and more.

**Live Site:** [**hub.mgks.dev**](https://hub.mgks.dev)

## About The Project

Automation Hub is a high-performance, SEO-optimized directory designed to help you find the perfect automation for any task. By aggregating community-sourced workflows from multiple platforms into a single, permanent index, we provide a unified search experience and cross-platform discovery.

### Key Features

* **Universal Multi-Platform Search:** Search across all supported platforms (n8n, OpenClaw, etc.) simultaneously.
* **Smart Categorization:** Discover related automations by tool, tag, or platform.
* **Daily Automated Discovery:** Our providers automatically scan GitHub for the latest community contributions.
* **Optimized for Speed:** A fast, static-site architecture built with Bun.

## Supported Platforms

- **n8n**: The powerful low-code tool to connect anything to everything.
- **OpenClaw**: Open-source automation skills and autonomous agents.
- *More coming soon...*

## How It Works

The platform uses a specialized static-site generator built with **Bun** that builds a fast, responsive interface from our permanent workflow database. Each source is managed as a pluggable provider, allowing us to rapidly add and index new automation tools as the community grows.
